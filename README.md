# Design and Implementation of a Lost and Found Management System for Institutional Environments-frontend

# Description
This is the frontend of the Lost & Found Management System.
It is built using Next.js and provides the user interface for reporting and searching lost and found items.

# Features 
User registration and login UI <br>
Report lost item form <br>
Report found item form <br>
Dashboard view <br>
Display matching results <br>

# Technologies Used
Next.js <br>
React <br>
